# Neo-pat-PEC Automation Extension

Using this extension, complete Neo-pat-PEC courses within seconds. It can be used for free to complete lectures, dialogue, reading materials, ungraded plugins, discussions, shareable link and course backup.

## Installation

1. Clone or download the repository.
2. Open the Chrome browser.
3. Go to `chrome://extensions/`.
4. Enable `Developer mode`.
5. Click on `Load unpacked`.
6. Select the downloaded folder.
7. The extension will be added to the browser.
